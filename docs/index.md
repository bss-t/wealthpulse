# Welcome to WealthPulse Docs

This documentation provides quick setup instructions, usage, and configuration notes for WealthPulse — an open-source expense management application.

## Getting Started

Follow the steps in the README to get the project running locally. For production, configure PostgreSQL and the appropriate environment variables.

## Features

- Expense tracking with receipts
- Budgets and reports
- Auto-categorization with ML
